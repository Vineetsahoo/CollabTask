# App Icons

This directory should contain your application icons:

- `icon.png` - 512x512 PNG icon for Linux
- `icon.ico` - Windows ICO file 
- `icon.icns` - macOS ICNS file

You can generate these from a high-resolution PNG using online tools or:
- For ICO: Use online converters or tools like ImageMagick
- For ICNS: Use `iconutil` on macOS or online converters

## Recommended sizes:
- PNG: 512x512 (will be resized as needed)
- ICO: Multiple sizes embedded (16, 24, 32, 48, 64, 128, 256)
- ICNS: Multiple sizes embedded (16, 32, 64, 128, 256, 512)

For now, the app will use default Electron icons until you add custom ones.
